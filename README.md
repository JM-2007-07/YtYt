# YtYt — Modern Landing Page

Современный адаптивный landing page, созданный в Figma с акцентом на минимализм, чистый UI и удобство пользователя.

---

##  About Project

Этот проект представляет собой современный веб-лендинг с продуманной структурой и адаптивным интерфейсом для разных устройств.

###  Основные особенности:
- Современный UI/UX дизайн
- Адаптивная верстка
- Минималистичный стиль
- Чистая типографика
- Удобная навигация
- Responsive Layout

---

##  Design

Макет разработан в **Figma** и включает:

- Desktop Version
- Tablet Version
- Mobile Version

---

##  Sections

###  Hero Section
Главный экран с акцентным заголовком и CTA-кнопкой.

###  About Section
Информация о продукте или сервисе.

###  Features Section
Блок с преимуществами и ключевыми возможностями.

###  Statistics Section
Карточки или показатели для усиления доверия пользователя.

###  Responsive Design
Полная адаптация под разные устройства.

###  Footer
Нижняя часть сайта с дополнительной навигацией.
